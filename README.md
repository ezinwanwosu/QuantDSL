# QuantDSL

QuantDSL is a domain-specific language (DSL) for expressing quantitative trading strategies in a concise and human-readable form. The project is written in Haskell and explores concepts from programming language design, functional programming, and quantitative finance.

Users can define trading rules such as moving-average crossovers and RSI-based signals, which are parsed into an Abstract Syntax Tree (AST), interpreted, and evaluated against historical market data through a backtesting engine.

This project serves as both a learning exercise in compiler/interpreter construction and an exploration of systematic trading strategy development.
